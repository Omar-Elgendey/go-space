 # Go Space 🚀  
### Coworking Spaces Finder Platform 🚀

## 📌 Overview

This project is a web-based platform that helps users discover and explore different coworking spaces such as *Shaghaf*, *Barah*, and *Workam*.
Users can view detailed information about each place including branches, services, amenities, and booking options.

The goal of this project is to make it easier for people to find the perfect workspace based on their needs.

---

## ✨ Features

* 🔍 Search for coworking spaces
* 📍 View different branches and locations
* 🛠️ Explore services and amenities
* 🧾 Detailed information about each workspace
* 📅 Booking system 
* 👤 User authentication (Login/Register)

## ⚙️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Omar-Elgendey/go-space.git
```

2. Navigate to the project folder:

```
cd go-space
```

3. Install dependencies:

```
pip install flask
```

4. Run the application:

```
python app.py
```

5. Open your browser and go to

```
http://127.0.0.1:5000/
```
